Detailed Description

API for touch input device object.

LameUI does NOT have its own Touch Input driver. User must supply their own functions to read touch input. Touch input data is stored in lui_touch_input_data_t:

typedef struct _lui_touch_input_data_s
{
uint8_t is_pressed;
int16_t x;
int16_t y;
}lui_touch_input_data_t;
