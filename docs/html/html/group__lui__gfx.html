Detailed Description

API for various graphics related functions like drawing line, rectangle, text etc. User does NOT require to use them. But if needed for some special reason, user may call these functions.

Note: lui_update() function will overwrite any manual gfx calls made by user. So, user must call gfx functions after the update function in a loop.

Advanced function to draw a string. For internal use only, but user may call it if needed. Most of the time draw_string_simple is enough for user.

This function lets user draw a string with specified fore color and background color. The background can be a bitmap image instead of color.

area sets string bounding box. When the area of the string is not known, width and height of area (area.w and area.h) can be set to 0. In this case, the function calculates the area. Width and height can be found using lui_gfx_get_string_dimension().

When the is_bg arg is 0, background color/image is not rendered, creating a tranaparent background. But the downside is, if the text is cahnged later, it cannot clear the previous pixels when is_bg is 0.

When is_bg is 1 and bg_bitmap is not NULL, the bitmap is rendered as background instead of bg_color. Set bitmap to NULL for rendering bg_color.

bitmap_crop argument is useful when the backgrounf bitmap image is bigger than the text area.

Parameters
strtext string
areaarea of the string. It sets drawing start positions and the dimension.
fore_colortext color
bg_colortext background color
bg_bitmaptext background bimap.
palettecolor palette for 1-bpp mono bitmap. Has no effect if bitmap is NOT 1-bpp.
bitmap_cropcrop area of the bitmap image.
is_bgflag decides whether to render background color/image or not
fontfont of the text

)

Get dimension of a string along x and y axis.

max_w (max width) sets the text's bounding box width. Past this width, text is wrapped and goes to next line. This value must not be bigger than the width of the display.

It does not return any value, rather modifies the str_dim array.

strtext
font_objfont
