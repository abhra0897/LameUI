Detailed Description

These are default icons used by LameUI. Not to be modified by user.

Icons can be used with strings like this: char* str = LUI_ICON_ARROW_FORWARD "My Home" LUI_ICON_HOME. Range of default icon index is 0x01 (1) to 0x1D (29). This MUST not be changed by user. User may add custom icons when creating font. Range of user defined icons is 0x80 (128) to 0xFF (255).

Note: Range 0x20 (32) to 0x7F (127) must be kept free for ASCII characters.
