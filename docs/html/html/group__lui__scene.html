Detailed Description

API for scene widget.

Scene is the main container of all widgets. It is at the top of hierarchy. Scene has no parent and must have at least one child. All widgets must be decedents of a scene.

An application may have multiple scenes. Only the active scene is rendered and it is set by lui_scene_set_active(lui_obj_t* obj_scene) function.
