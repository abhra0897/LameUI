◆ lui_bitmap_mono_pal_t

typedef struct _lui_bitmap_mono_pal_s lui_bitmap_mono_pal_t

Color palette for 1-bpp mono bitmap image.

This color palette has NO effect on 8-bpp grayscale and 16-vpp rgb565 bitmaps.

◆ lui_bitmap_t

typedef struct _lui_bitmap_s lui_bitmap_t

bitmap datatype. Used to render images. Generated using LameUI font tools

Bitmap must be generated with LameUI tools (see docs). Supported bits-per-pixel (bpp) are 1, 8, and 16.

lui_obj_t contains all the common data of objects along with address of the extended data as per the type of the object
