Set data source of the line chart.

Data source contains both x and y data. Odd index: X, Even index: 1. That means, data source array looks like: {x0, y0, x1, y1, ..., xn, yn}

◆ lui_linechart_set_draw_mode()

void lui_linechart_set_draw_mode ( lui_obj_t *  obj_linechart,
uint8_t  mode_flag 
)

Set draw mode of line chart's plot line Sets whether to draw line segments, or only the points/markers, or both.

flags: LUI_LINECHART_DRAW_MODE_LINE -> draw line segments by connecting points LUI_LINECHART_DRAW_MODE_POINT -> draw points/markers

Flags can be ORed to enable both of them.

Parameters
obj_linechartlinechart object
mode_flagORed drawmode flags.
