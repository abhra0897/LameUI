Detailed Description

API for linechart widget.

Example

Dark Theme
Light Theme
double fib_data[40] = {0,0, 1,1, 2,1, 3,2, 4,3, 5,5, 6,8, 7,13, 8,21, 9,34};
lui_obj_t* my_chart = lui_linechart_create();
lui_linechart_set_data_source(my_chart, fib_data, 10);
lui_linechart_set_grid_count(my_chart, 4, 2);
lui_linechart_set_line_color(my_chart, lui_rgb(255, 0, 0));
// Setting line chart's area is important!
lui_object_set_area(my_chart, 120, 80);

Set whether to apply automatic scaling on the data source or not. If auto scale is enabled, graph will always be in the boundary of drawing area. If auto scale is disabled, user must provide manual range.

Parameters
obj_linechartlinechart object
auto_scale1: enabled (default), 0: disabled

double *  source,
uint16_t  points 
)

Set data source of the line chart.

Data source contains both x and y data. Odd index: X, Even index: 1. That means, data source array looks like: {x0, y0, x1, y1, ..., xn, yn}

Parameters
obj_linechartlinechart object
sourcearray of data points (in x,y format)
pointsnumber of data points
