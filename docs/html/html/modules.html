API Documentation
These are the API of LameUI
[detail level 123]
 ▼API FunctionsThis group of functions provide access to LameUI core. All objects are accessed by getter and setter functions
 Core initialize & update APIThese functions are used to initialize the core and to update it
 All objects (widgets) APIThese functions are applicable for all types of objects. Only for a few particular types of objects, one or more of these functions have no effect
 Label APIAPI for label widgets
 Linechart APIAPI for linechart widget
 Button APIAPI for button widget
 Switch APIAPI for switch widget
 Checkbox APIAPI for checkbox widget
 Slider APIAPI for slider widget
 List APIAPI for list widget
 Buttongrid APIAPI for buttongrid widget
 Keyboard APIAPI for Keyboard widget
 Textbox APIAPI for textbox widget
 Panel APIAPI for panel widget
 Scene APISet background image of the panel
 Display Driver APISet background bitmap image of a scene
 Input Device APIAPI for touch input device object
 Graphics APIAPI for various graphics related functions like drawing line, rectangle, text etc. User does NOT require to use them. But if needed for some special reason, user may call these functions
 ▼Macro DefinitionsThis is the group of all public macro definitions. Public macros always start with LUI_ prefix
 ▼User ConfigUser may change configuration using these macros. All these macros can be found in lame_ui.h file
 Widgets to useDefines which widgets to use. Only if the relevant macro is defined, that widget will be compiled. Comment out the object that you don't want to use. This helps to save Flash
 State TypesAll input states of LameUI
 Event TypesAll input events of LameUI. Events depend on previous and current state
 Object (widget) TypesAll types of objects (widgets) in LameUI
 Keyboard modesKeyboard modes of LameUI
 Default built-in iconsThese are default icons used by LameUI. Not to be modified by user
 Buttongrid button property-byte masksProperty byte masks of a buttongrid. Masks set different properties of a button in a buttongrid
 Content alignment flagsContent alignment flags
 Slider knob type flagsKnob type flag for slider widget. It defines how the knob of a slider is rendered
 LameUI default fontThis is default font of LameUI. User can access default font using this macro
